n8n SSH 크레덴셜 설정 가이드
PC 서버 (WSL2) 연결 방법
📋 Step 1 — WSL2에서 SSH 서버 활성화 
# WSL2 Ubuntu에서 실행
sudo apt update
sudo apt install openssh-server -y

# SSH 서비스 시작
sudo service ssh start

# 자동 시작 설정
echo "sudo service ssh start" >> ~/.bashrc

# 포트 확인 (기본 22)
sudo netstat -tlnp | grep :22
📋 Step 2 — n8n SSH 크레덴셜 등록 
# n8n 크레덴셜 설정값
Type    : SSH
Host    : 127.0.0.1
         (n8n이 같은 PC면 localhost)
         (VPS에서 접근 시 CF Tunnel 주소)
Port    : 22
Username: [WSL2 사용자명]
Auth    : Password 또는 Private Key

# Private Key 방식 (권장)
ssh-keygen -t ed25519 -C "n8n-coching"
cat ~/.ssh/id_ed25519.pub >> ~/.ssh/authorized_keys
# id_ed25519 내용을 n8n에 붙여넣기
📋 Step 3 — 프로젝트 디렉토리 구성 
# PC WSL2에서 실행
mkdir -p ~/coching-server/{scripts,prompts,logs,db}

# 스크립트 복사 (이 패키지에서)
cp scripts/update-db.sh ~/coching-server/scripts/
cp prompts/*.md ~/coching-server/prompts/
chmod +x ~/coching-server/scripts/update-db.sh

# Claude Code 설치 확인
claude --version
# 없으면:
npm install -g @anthropic-ai/claude-code

# 테스트 실행
cd ~/coching-server
bash scripts/update-db.sh inci
⚡ n8n SSH 노드 Command 설정 
cd ~/coching-server && bash scripts/update-db.sh {{ $json.target }} 2>&1
