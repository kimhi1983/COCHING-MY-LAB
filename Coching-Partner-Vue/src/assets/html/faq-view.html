원료 등록하는 방법
운영자16:24

원료를 등록하는 방법은 다음과 같습니다.

1. 원료등록 및 관리 페이지 접속
2. 등록하기 버튼 클릭
3. 원료정보 입력 성분정보
· 1개 입력시 단일성분 성분정보
· 2개 이상 입력시 복합성분
4. 자료 작성 후 저장버튼 클릭
